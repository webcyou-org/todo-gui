# ToDo - gtk-rs

Rust + [gtk-rs](https://gtk-rs.org/) (GTK4) で実装した ToDo アプリケーション。

## Requirements

- Rust (stable)
- GTK4 development libraries

### Ubuntu / Debian

```bash
sudo apt-get install libgtk-4-dev
```

### macOS

```bash
brew install gtk4
```

## Build & Run

```bash
cd Rust/gtk-rs/todo
cargo run
```
