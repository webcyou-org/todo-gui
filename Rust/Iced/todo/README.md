# Iced Todo

Rust Todo app using [Iced](https://iced.rs/) 0.12 — cross-platform GUI library inspired by Elm.

## Requirements

- Rust (stable) 1.70+

```sh
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

## Build & Run

```sh
cargo run
```

本番ビルド:

```sh
cargo build --release
./target/release/todo
```
