# React Native Todo

TypeScript Todo app using [React Native](https://reactnative.dev). Supports iOS, Android, and macOS.

## Requirements

- Node.js 18+
- macOS: Xcode installed
- iOS/Android: [React Native environment setup](https://reactnative.dev/docs/environment-setup) must be completed

**macOS**: `brew install node`
**Windows**: `winget install OpenJS.NodeJS`
**Linux (Ubuntu/Debian)**: `sudo apt install nodejs npm`

## Install

```sh
npm install

# For iOS (first time only)
cd ios && pod install && cd ..
```

## Run

```sh
# Start Metro bundler
npm start

# Launch app in a separate terminal
npm run macos    # macOS
npm run ios      # iOS Simulator
npm run android  # Android Emulator
```

## Architecture

React Native declarative component structure. TypeScript separates the data model and components, with `App.tsx` defining the root component. A single codebase supports iOS, Android, and macOS.

```
.
├── App.tsx             # root component · state management
└── src/
    ├── data/           # Todo data model · TabFilter
    ├── components/     # components for input, tabs, and list
    └── theme.ts        # color and style constants
```
