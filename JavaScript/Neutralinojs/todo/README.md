# Neutralinojs Todo

JavaScript Todo app using [Neutralinojs](https://neutralino.js.org/) — lightweight desktop app framework using the system WebView (no bundled browser engine).

## Requirements

- Node.js 18+
- Neutralinojs CLI

```sh
npm install -g @neutralinojs/neu
```

## Install

```sh
neu update
```

## Run

```sh
neu run
```

## Build（配布バイナリ）

```sh
neu build
# 出力先: dist/
```
