# FLTK Todo

C++ Todo app using [FLTK](https://www.fltk.org/) (Fast Light Toolkit).

## Requirements

- CMake 3.15+
- FLTK

## Install

```sh
brew install cmake fltk
```

## Build

```sh
cmake -B build
cmake --build build
```

## Run

```sh
open build/main.app
```

> macOS でビルドすると `.app` バンドルが生成されます。
