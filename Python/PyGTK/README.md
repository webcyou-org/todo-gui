# PyGTK (PyGObject) Todo

Python Todo app using [PyGObject](https://pygobject.gnome.org/) (GTK3 bindings for Python).

## Requirements

- Python 3.9+
- GTK3
- gobject-introspection

```sh
brew install gtk+3 gobject-introspection pygobject3
```

## Install

```sh
pip install PyGObject
```

## Run

```sh
python main.py
```
