# Reflex Todo

Python Todo app using [Reflex](https://reflex.dev/) — full-stack web framework in pure Python.

## Requirements

- Python 3.9+
- Node.js 18+ (Reflex のフロントエンドビルドに使用)

## Install

```sh
pip install reflex
```

## Initialize (初回のみ)

```sh
reflex init
```

## Run

```sh
reflex run
```

ブラウザで `http://localhost:3000` が自動的に開きます。
