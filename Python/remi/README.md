# remi Todo

Python Todo app using [remi](https://github.com/rawpython/remi) — browser-based GUI toolkit.

アプリ起動後、ブラウザで `http://localhost:8080` を開いてください。

## Requirements

- Python 3.9+

## Install

```sh
pip install remi
```

## Run

```sh
python main.py
```

ブラウザが自動的に起動します。起動しない場合は `http://localhost:8080` にアクセスしてください。
