# Wails Todo

Go + Vue3 Todo app using [Wails](https://wails.io/) v2 — build desktop apps with Go backend and web frontend.

## Requirements

- Go 1.21+
- Node.js 18+
- Wails CLI

```sh
go install github.com/wailsapp/wails/v2/cmd/wails@latest
```

**macOS** (requires Xcode Command Line Tools):
```sh
xcode-select --install
brew install go node
```

**Windows**
```sh
winget install GoLang.Go OpenJS.NodeJS
```

**Linux (Ubuntu/Debian)**
```sh
sudo apt install golang-go nodejs npm
```

## Install

```sh
wails doctor   # environment check
```

## Run (development mode)

```sh
wails dev
```

Frontend changes are hot-reloaded. Go code can also be called from browser devtools at `http://localhost:34115`.

## Build (distribution binary)

```sh
wails build
# Output: build/bin/
```

## Architecture

A two-layer structure with a Go backend and a Vue3 frontend. Wails provides an IPC bridge so that Go methods can be called transparently from the frontend.

```
.
├── main.go         # entry point · Wails app launch
├── app.go          # Go backend (Todo CRUD methods · state management)
├── frontend/       # Vue3 + Vite frontend UI
│   └── src/        # App.vue · components · styles
└── wailsjs/        # Go→JS bindings auto-generated by Wails
```
