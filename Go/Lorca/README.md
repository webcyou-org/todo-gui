# Lorca Todo

Go Todo app using [Lorca](https://github.com/zserge/lorca) — builds desktop apps with Go and HTML/CSS, using Chrome as the UI layer.

## Requirements

- Go 1.20+
- Google Chrome または Chromium インストール済み

```sh
brew install go
brew install --cask google-chrome
```

## Install

```sh
go mod download
```

## Run

```sh
go run .
```
