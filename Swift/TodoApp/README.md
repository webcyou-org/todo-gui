# SwiftUI Todo

Swift Todo app using [SwiftUI](https://developer.apple.com/xcode/swiftui/) — Apple's declarative UI framework for macOS.

## Requirements

- macOS 14+
- Xcode Command Line Tools または Xcode

```sh
xcode-select --install
```

## Build

```sh
swift build
```

## Run

```sh
swift run
```

または:

```sh
.build/debug/TodoApp
```
